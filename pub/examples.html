PermForm.js Demo

Here, we use PermForm.js to create a simple job application. By using PermForm.js, we can create this form with a simple JSON string.
The JSON for the form is on the left, and the rendered form is shown on the right.
The key features of the form include:

  1. Automatic grid alignment (each element is defined using a coordinate system)
  2. Auto save (close and reopen this tab, the form will save its state!)
  3. Input validation (the phone number field is set to be numeric type, the email is validated to be an email, and all the fields are set to be required).
